#  EEG-Based Emotion Recognition

##  Project Overview

This project focuses on detecting human emotions using EEG (Electroencephalogram) signals. By analyzing brainwave data, the model predicts emotional states, which can be useful in applications like mental health monitoring, human-computer interaction, and affective computing.

---

##  Objective

* Process EEG signal data
* Extract meaningful features
* Train machine learning models
* Predict emotional states based on brain activity

---

##  Dataset

The dataset consists of EEG signal recordings stored in `.dat` files.

 Due to large file size, the dataset is not included in this repository.
You can access it from the original source (example):
 https://www.eecs.qmul.ac.uk/mmv/datasets/deap/

##  How to Run the Project

1. Clone the repository:

```
git clone https://github.com/devajay4u-coder/EEG-Based-Emotion-Recognition.git
```

2. Navigate to the project folder:

```
cd EEG-Based-Emotion-Recognition
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Open the notebook:

```
jupyter notebook
```

5. Run the file:

```
EEG_data (1).ipynb
```
